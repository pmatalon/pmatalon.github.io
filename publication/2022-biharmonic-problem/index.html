Iterative solution to the biharmonic equation in mixed form discretized by the Hybrid High-Order method

P. Antonietti, P. Matalon, M. Verani
2022
Type
Preprint
Publication
Working paper

Reproduction of the numerical experiments

The experiments can be reproduced through the open-source code fhhos4, release 1.2.

Convergence of the discrete normal derivative

-pb diff -geo square -source sine -s ch -mesh tri -normalder -k 0 -n 256

Convergence of the biharmonic scheme

Square, Cartesian mesh: use the arguments

-pb bihar -geo square -source poly -s ch -bihar-prec s -nbh-depth 8 -mesh cart -cs r -kc 1

k=0:

-k 0 -n  16 -tol 1e-7
-k 0 -n  32 -tol 1e-7
-k 0 -n  64 -tol 1e-7
-k 0 -n 128 -tol 1e-7
-k 0 -n 256 -tol 1e-7

k=1:

-k 1 -n  16 -tol 1e-7
-k 1 -n  32 -tol 1e-7
-k 1 -n  64 -tol 1e-7
-k 1 -n 128 -tol 1e-7
-k 1 -n 256 -tol 1e-7

k=2:

-k 2 -n  16 -tol 1e-7
-k 2 -n  32 -tol 1e-7
-k 2 -n  64 -tol 1e-7
-k 2 -n 128 -tol 1e-9
-k 2 -n 256 -tol 1e-11

k=3:

-k 3 -n  16 -tol 1e-7
-k 3 -n  32 -tol 1e-8
-k 3 -n  64 -tol 1e-10
-k 3 -n 128 -tol 1e-11
-k 3 -n 256 -tol 1e-13 -bihar-prec s

Preconditioner convergence

-pb bihar -geo square -source poly -s ch -mesh cart -cs r -k 1 -n 256 -tol 1e-14 -export iter -bihar-prec no #no preconditioner
-pb bihar -geo square -source poly -s ch -mesh cart -cs r -k 1 -n 256 -tol 1e-14 -export iter -bihar-prec p -nbh-depth {2|4|6|8|10} #with preconditioner

Heuristics

-pb bihar -geo square -source poly -s fcguamg -mesh tri -cs r -k 3 -n 32 -tol 1e-8 -bihar-prec p -nbh-depth 2 -export iter -opt2 {0|2}
